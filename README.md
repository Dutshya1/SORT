# Организатор файлов (File Organizer)

Python-скрипт для автоматической сортировки файлов в директории по их расширениям.

## Сайт программы
  
 https://dutshya1.github.io/SORT/ 

## Возможности

- **Автоматическая сортировка**: Перемещает файлы в соответствующие папки:
  - `Images` (Изображения)
  - `Documents` (Документы)
  - `Audio` (Аудио)
  - `Video` (Видео)
  - `Archives` (Архивы)
- **Неизвестные файлы**: Файлы с нераспознанными расширениями перемещаются в папку `Others`.
- **Обработка дубликатов**: Если файл с таким именем уже существует в целевой папке, скрипт автоматически переименовывает новый файл (например, `photo_1.jpg`).
- **Логирование**: Выводит информацию о перемещенных файлах в консоль в реальном времени.
- **Безопасность**: Не перемещает директории и сам файл скрипта.

## Требования

- Python 3.6+

## Использование

1. **Скачайте** файл `file_organizer.py`.
2. **Запустите скрипт**:
   ```bash
   python file_organizer.py
   ```
3. **Укажите путь к директории**:
   - Вам будет предложено ввести полный путь к папке, которую нужно отсортировать.
   - Нажмите **Enter**, не вводя ничего, чтобы отсортировать *текущую* директорию, в которой находится скрипт.

## Настройка

Вы можете настроить распределение файлов по папкам, отредактировав словарь `extensions_map` в файле `file_organizer.py`:

```python
self.extensions_map = {
    'Images': ['.jpg', '.png', ...],
    # Добавьте свои категории:
    'Code': ['.py', '.js', '.html']
}
```
